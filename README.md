# ⚡ NovaTech - Loja Online de Tecnologia

Uma loja online moderna e responsiva de tecnologia, construída com HTML, CSS e JavaScript puros.

## 🚀 Features

- **Design Premium Dark Mode** - Interface moderna com glassmorphism e gradientes
- **Carrinho de Compras** - Adicione, remova e gerencie itens com persistência (localStorage)
- **Busca Inteligente** - Pesquise produtos em tempo real (Ctrl+K)
- **Lista de Desejos** - Salve seus produtos favoritos
- **Filtros de Categoria** - Filtre produtos por categoria
- **Countdown de Ofertas** - Timer regressivo para promoções
- **Formulário de Contato** - Seção completa de contato
- **Newsletter** - Captura de e-mails
- **Animações Suaves** - Micro-animações e scroll reveal
- **100% Responsivo** - Funciona em todos os dispositivos
- **SEO Otimizado** - Meta tags e semântica HTML5

## 🛠️ Tecnologias

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript ES6+
- [Lucide Icons](https://lucide.dev/)
- [Google Fonts](https://fonts.google.com/) (Inter & Space Grotesk)

## 📦 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/SEU-USUARIO/loja-online.git
```

2. Abra o `index.html` no navegador ou use um server local:
```bash
npx serve .
```

## 🌐 Deploy

Este projeto está pronto para deploy no **GitHub Pages**:
1. Vá em Settings > Pages
2. Selecione a branch `master` e a pasta `/` (root)
3. Salve e aguarde o deploy
